# 📘 要件仕様書：Study Hour Calendar（学習時間カレンダー）

## 1. 概要

### 1.1 システム名

**Study Hour Calendar（学習時間カレンダー）**

### 1.2 目的

学習時間を記録・可視化し、学習習慣を定着させる。\
ユーザーが勉強時間でカレンダーを埋め、達成状況を視覚的に確認できる仕組みを提供する。

### 1.3 対象ユーザー

-   学生（中高生〜大学生）
-   社会人（資格試験・自己啓発）
-   自己管理を目的とした学習者

------------------------------------------------------------------------

## 2. 機能要件

### 2.1 予定機能（Plan Function）

**概要：** その日の予定勉強時間を設定できる。

**詳細仕様：** - 日ごとに「予定勉強時間（hour / min）」を登録可能。 -
予定時間はカレンダー上に自動的に反映される。 -
週・月・年単位での合計予定時間を自動算出。

------------------------------------------------------------------------

### 2.2 記録機能（Record Function）

**概要：** 勉強時間を記録ボタンで簡単に登録する。

**詳細仕様：** - 日付は固定（今日の日付を自動取得）。 -
記録ボタンを押すと、以下の時間選択肢が表示される： -
10分、20分、30分、40分、50分、1時間 -
チェックを入れた時間を記録ボタンで登録。 -
または、ユーザーが任意の分数を入力して記録する欄も用意。 -
「何時間、何分を記録するか」を自由に指定可能。

------------------------------------------------------------------------

### 2.3 達成率計算機能（Achievement Percentage Function）

**概要：**
予定勉強時間に対する実際の勉強時間の達成率（%）を算出・表示する。

**詳細仕様：** - 日単位：当日の予定勉強時間をどれだけ達成したか（%） -
週単位：その週の合計予定勉強時間をどれだけ達成したか（%） -
月単位：その月の合計予定勉強時間をどれだけ達成したか（%） -
年単位：その年の合計予定勉強時間をどれだけ達成したか（%） -
任意期間（xx/xx〜xx/xx）で達成率を確認できる。

------------------------------------------------------------------------

### 2.4 カレンダー機能（Calendar Function）

**概要：** 記録データをカレンダーに色で可視化する。

**詳細仕様：** -
1日の記録を集計し、その日の達成率に応じてカレンダーに色付け。 -
達成率ごとの色分布（デフォルト設定）：

  達成率      表示色
  ----------- --------
  0〜49%      白
  50〜59%     黄
  60〜69%     緑
  70〜79%     茶
  80〜89%     青
  90〜99%     黒
  100%        紫
  110〜119%   黒
  120〜129%   紫
  130〜139%   緑
  140〜149%   白
  150%〜      白

※色設定は将来的にカスタマイズ可能にする。

------------------------------------------------------------------------

## 3. 非機能要件

  ----------------------------------------------------------------------------------------------
  項目                                要件
  ----------------------------------- ----------------------------------------------------------
  **パフォーマンス**                  日ごとの記録操作は1秒以内に反映されること

  **データ保存**                      ローカルストレージまたはユーザーアカウントに紐づくDB保存

  **UI/UX**                           シンプルかつ色の変化で直感的に理解できるデザイン

  **拡張性**                          週・月・年単位でグラフ表示を追加できる設計

  **セキュリティ**                    学習データはユーザー本人以外に閲覧できないこと
  ----------------------------------------------------------------------------------------------

-   目標時間自動調整AI（過去実績から最適な予定時間を提案）
